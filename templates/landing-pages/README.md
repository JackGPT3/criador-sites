# Templates de Landing Pages

Templates prontos para copiar e adaptar. Cada pasta tem um `index.html` completo e funcional.

## Tipos

| Pasta | Quando usar |
|---|---|
| `vendas/` | Produto ou serviço com preço definido. Seções: hero, benefícios, depoimentos, o que inclui, preço, FAQ, CTA |
| `lead-gen/` | Captura de e-mail ou WhatsApp. Formulário em destaque, sem preço. |
| `lancamento/` | Curso, mentoria, evento com countdown. Tom de urgência e escassez. |

## Como usar

1. Copiar a pasta inteira para `clientes/[cliente]/[nome-projeto]/`
2. Editar os valores das variáveis CSS no `:root` (cores, fontes)
3. Seguir os comentários `<!-- EDITAR: ... -->` no HTML
4. Substituir textos placeholder pelo conteúdo real
5. Colocar imagens em `assets/images/` e atualizar os `src`
