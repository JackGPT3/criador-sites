# Biblioteca de Componentes

Blocos HTML isolados e reutilizáveis. Cada arquivo tem HTML + CSS (e JS quando necessário) autocontido — pronto para colar em qualquer projeto.

## Como usar

Copiar o bloco inteiro (HTML + `<style>`) para dentro do seu projeto.
Mover o CSS para o arquivo `style.css` do projeto.
Seguir os comentários `<!-- EDITAR: ... -->`.

## Componentes disponíveis

> Pasta vazia por enquanto. Use `/componente [tipo]` para gerar e salvar componentes aqui.

## Tipos suportados pelo `/componente`

- `hero` — seção de abertura com headline e CTA
- `header` — cabeçalho com nav e logo
- `cta` — seção chamada para ação isolada
- `formulario` — form de contato ou lead gen
- `depoimentos` — grid de testemunhos
- `faq` — accordion de perguntas frequentes
- `beneficios` — grid de features/diferenciais
- `rodape` — footer completo
- `preco` — card ou tabela de preço
- `contador` — countdown timer
- `numeros` — seção de estatísticas
- `video` — embed de YouTube/Vimeo
