Você abre o ChatGPT, digita "me ajuda a escrever um e-mail para um cliente" e recebe uma resposta genérica que você não usaria nunca. Joga fora, tenta de novo, mais genérico ainda. Fecha a aba achando que "essa IA não serve pra nada."

O problema quase nunca é a IA. É o prompt.

A diferença entre uma resposta inútil e uma que você copia direto para o cliente está em como você perguntou. Isso tem uma lógica — e dá pra aprender em menos de 20 minutos.

## O que é um prompt, de verdade

Prompt é a instrução que você dá para a IA. Não é diferente de pedir algo a um funcionário novo no primeiro dia: quanto mais contexto você der, menos vai precisar corrigir depois.

"Me faz um e-mail" é como dizer "faz alguma coisa aí". "Me faz um e-mail de follow-up para uma cliente de consultoria que pediu proposta há 3 dias e ainda não respondeu, tom profissional mas direto, sem ser insistente" — isso tem contexto suficiente para gerar algo usável.

A boa notícia: você não precisa virar especialista em "engenharia de prompt" para ter bons resultados. Precisa entender uma estrutura simples.

## A estrutura que funciona: Contexto + Tarefa + Formato + Restrições

Todo prompt eficiente tem esses quatro elementos. Nem sempre você precisa dos quatro, mas quando a resposta não está boa, é porque algum deles está faltando.

**Contexto** — quem você é, qual é a situação, para quem é o resultado.

**Tarefa** — o que você quer que a IA faça, sem ambiguidade.

**Formato** — como você quer receber: lista, parágrafo, tabela, e-mail, roteiro.

**Restrições** — o que não pode aparecer, tom de voz, tamanho, limitações.

Exemplo com a estrutura completa:

> *Contexto:* Sou dono de uma loja de roupas femininas em Belo Horizonte. Tenho uma cliente VIP que compra todo mês e não apareceu nos últimos 45 dias.
>
> *Tarefa:* Escreva uma mensagem de WhatsApp para reconectar com ela.
>
> *Formato:* Mensagem curta, máxima 3 linhas, tom pessoal.
>
> *Restrições:* Não mencione desconto. Não pareça desesperado. Só deixe a porta aberta para ela responder quando quiser.

Com esse prompt, a IA entrega algo que você usa. Sem ele, você recebe um texto de call center.

## 8 exemplos de prompts para situações reais

### 1. Responder reclamação de cliente

> Sou dono de uma lanchonete. Um cliente deixou uma avaliação negativa no Google dizendo que o pedido chegou frio e o atendente foi grosseiro. Escreva uma resposta pública para essa avaliação: reconheça o problema sem concordar com tudo, peça desculpas de forma genuína e ofereça uma solução. Máximo 4 linhas, tom profissional e humano.

### 2. Criar legenda para Instagram

> Tenho uma clínica de estética em São Paulo. Vou postar uma foto do antes e depois de um tratamento de pele. Escreva 3 opções de legenda para Instagram: uma mais emocional, uma mais informativa e uma com humor leve. Inclua até 5 hashtags relevantes em cada uma.

### 3. Fazer análise de concorrente

> Segue o texto retirado do site de um concorrente meu na área de assessoria contábil: [cole o texto]. Liste os principais argumentos de venda que ele usa, o que parece ser o público-alvo e quais pontos fracos você identifica na comunicação dele.

### 4. Criar proposta comercial

> Sou consultor de marketing digital. Preciso enviar uma proposta para uma academia de ginástica que quer atrair mais alunos pelo Instagram. O projeto inclui gestão de perfil por 3 meses, produção de 12 posts por mês e um relatório mensal. O valor é R$1.800/mês. Escreva uma proposta comercial de 1 página, tom profissional, que destaque resultado e não processo.

### 5. Criar FAQ para site

> Tenho uma clínica de psicologia online que atende pelo plano de saúde e particular. Crie 8 perguntas frequentes com respostas para colocar no site. Foco em dúvidas reais de quem nunca fez terapia antes. Respostas curtas, sem jargão clínico.

### 6. Analisar planilha ou texto financeiro

> Segue o resumo das minhas despesas do mês passado: [lista de despesas]. Identifique as 3 categorias que mais pesam no orçamento, aponte algo que parece fora do padrão e sugira onde posso cortar sem impactar a operação.

### 7. Escrever descrição de produto

> Tenho uma loja virtual de suplementos. Preciso de uma descrição para este produto: Whey Protein Concentrado 900g, sabor baunilha, 24g de proteína por dose, sem lactose. Escreva para o público de mulheres que treinam 3x por semana e não são atletas profissionais. Máximo 80 palavras, tom acessível.

### 8. Criar roteiro para vídeo curto

> Quero fazer um Reels de 30 segundos para a minha marcenaria. O objetivo é mostrar que fazemos móveis sob medida em qualquer tamanho de apartamento. Crie um roteiro com narração e sugestão de visual para cada cena. Tom descontraído, não pareça propaganda.

## Os 4 erros mais comuns

**1. Prompt vago demais**

"Me ajuda com marketing" não é um prompt. A IA não sabe se você quer um post, uma estratégia, um orçamento ou uma análise. Quanto mais específico, melhor o resultado.

**2. Não dizer o formato esperado**

Se você quer uma tabela, diga "em formato de tabela". Se quer uma lista numerada, diga isso. A IA vai tentar adivinhar e frequentemente escolhe o formato errado.

**3. Não dar contexto do negócio**

"Escreva um e-mail de vendas" e "escreva um e-mail de vendas para uma oficina mecânica em cidade pequena, público de agricultores, tom simples e direto" são pedidos completamente diferentes. O segundo tem contexto. O primeiro não.

**4. Aceitar a primeira resposta sem refinar**

O ChatGPT raramente acerta de primeira quando o pedido é complexo. Trate como uma conversa: diga o que ficou bom, o que precisa mudar, e peça a versão revisada. "Gostei do tom, mas quero mais curto e sem o último parágrafo" já melhora muito.

## Como melhorar um prompt que não funcionou

Quando a resposta não está boa, faça essas perguntas antes de tentar de novo:

- Faltou contexto sobre quem eu sou ou para quem é?
- O formato esperado estava claro?
- Eu defini o tom de voz?
- Tinha alguma restrição que eu não mencionei?

Depois corrija e envie de novo na mesma conversa. Você não precisa começar um chat novo — a IA lembra o que foi dito antes e você pode pedir ajustes em cima da resposta anterior.

Uma técnica útil: **peça para a IA te ajudar a melhorar o prompt**. Escreva o que você quer fazer e pergunte: "Como eu reescreveria esse prompt para ter um resultado melhor?" Funciona bem quando você sabe que o pedido está confuso mas não sabe como melhorar.

## Perguntas frequentes

**O ChatGPT vai dar respostas diferentes para o mesmo prompt?**

Sim. Ele tem um componente aleatório por design. Se você receber uma resposta ruim para um bom prompt, tente de novo — às vezes a segunda ou terceira tentativa é melhor. Você pode pedir explicitamente "me dê 3 versões diferentes" para comparar.

**Preciso escrever um prompt longo para ter bom resultado?**

Não necessariamente. Um prompt de duas linhas bem construído entrega mais que um de dez linhas confuso. O tamanho certo é o necessário para eliminar ambiguidade — nem mais, nem menos.

**Posso usar o mesmo prompt sempre para a mesma tarefa?**

Pode, e é recomendado. Quando você encontrar uma estrutura que funciona bem para uma tarefa recorrente — responder e-mails, criar legendas, escrever propostas — salve em um documento. Vira um ativo da sua operação. Com o tempo, você tem uma biblioteca de prompts calibrados para o seu negócio.
